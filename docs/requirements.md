# Poziadavky

## Funkcie
- Autonomne riadenie technologii studne (PLC)
- Monitorovanie, diagnostika a lokalne ovladanie (ESP monitor)
- Remote pristup a sprava (Supervisor)
- Fail-safe spravanie pri chybach a vypadkoch

## Bezpecnost
- Bezpecne stavy pri poruche senzorov alebo akcie
- Ochrana proti suchobehu a prehriatiu
- Overenie zopnutia styka cov/isticov

## Komunikacia
- Stabilna komunikacia PLC <-> ESP monitor
- Prenos telemetrie a alarmov
- Moznost lokalneho aj vzdialeneho ovladania
